# ADR-010 — Use Git as the canonical configuration registry

Status: Proposed  
Date: 2026-07-07

---

## Context

Prompt, role, router and policy changes need review, diff, rollback and auditability.

---

## Decision

Git will be the canonical registry for prompt-related configuration, documentation, schemas and ADRs.

---

## Consequences

The project keeps configuration changes reviewable and compatible with GitHub/GitLab workflows.
